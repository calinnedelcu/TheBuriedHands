# AI Start Here

Acesta este fișierul de intrare pentru orice AI / agent / colaborator care deschide proiectul într-un chat nou.

## Ce este jocul

**TheBuriedHands** este un joc stealth low-poly plasat în mausoleul lui Qin Shi Huang în 210 î.Hr., unde joci un meșteșugar prins în timpul sigilării mormântului, încercând să scape, să salveze un ucenic sau să trimită adevărul afară înainte ca porțile să se închidă pentru totdeauna.

## Ce trebuie înțeles imediat

- Nu este fantasy.
- Nu este un joc de luptă arcade.
- Nu este despre un erou care schimbă istoria.
- Este despre tensiune, supraviețuire, stealth, puzzle-uri mecanice și tragedie istorică.
- Interiorul real al mormântului central nu este cunoscut complet, deci jocul prezintă un sector istoric-plauzibil, nu o hartă reală declarată.

Formulare canonică:

> Reconstrucție istoric-plauzibilă inspirată de descrierile lui Sima Qian și de descoperirile arheologice ale complexului Qin Shi Huang.
